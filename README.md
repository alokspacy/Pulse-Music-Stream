# 📖 Overview

**Pulse Music Stream** is a premium, high-fidelity music streaming platform branded and developed by **Alok Singh**. The platform allows users to search and stream millions of high-quality tracks directly from the web, build and manage custom playlists, save albums, and follow favorite artists using real-time data fetched from the **Deezer API**. 

The system leverages **Auth.js (v5)** for user credentials and Google OAuth authentication, **Neon PostgreSQL** managed by the **Prisma ORM** for persistent user libraries, and features a state-of-the-art interactive audio equalizer alongside a live soundwave visualizer spectrum that aligns perfectly with a rotating vinyl disc playing experience.

---

# ✨ Features

| Feature | Description |
|:---|:---|
| 🎵 **Real-time Catalogue Streaming** | Search, stream, and explore charts for millions of songs, albums, and artists, powered directly by the Deezer Music API. |
| 🎚️ **Interactive Audio Equalizer** | Adjust sound profile frequencies (60Hz, 150Hz, 400Hz, 1kHz, 3kHz, 8kHz, 15kHz) dynamically using a custom HTML5 Audio node equalizer. |
| 👁️ **Live Audio Visualizer** | A canvas visualizer displaying dynamic ambient glows and symmetric equalizer spectrum rings aligned perfectly with a rotating vinyl record. |
| 🔑 **Auth.js Credentials & Google OAuth** | Secure user login and registration supporting email credential accounts and instant Google login. |
| 📂 **Library Management** | Personal account library allowing users to create playlists, bookmark liked tracks, follow artists, and save albums. |
| 🐘 **Neon cloud Database** | Robust, cloud-based data storage and retrieval utilizing Neon serverless PostgreSQL. |
